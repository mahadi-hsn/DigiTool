# DigiTools

A modern and responsive web app for exploring digital tool packages, checking pricing plans, and selecting the right package for your needs.

## Tech Stack

- React 19
- Vite
- Tailwind CSS + DaisyUI
- Lucide React Icons
- React Toastify

## Features

- Browse multiple tool packages with title, pricing, billing type, badge, and detailed feature lists.
- Add/select packages with instant user feedback using toast notifications.
- Clean, responsive UI sections including package cards, pricing cards, steps, and reusable components.
